    Save Files & Editors

    Starting over in Stardew Valley? Use these tools to set up your perfect game without spending hours grinding.

    🎮 Web Save File Editor

    Open Stardew Valley Save Editor (free, runs in browser)

    This online tool lets you safely edit your save file:

    Feature What You Can Change
    🧑 Player Name, appearance, gender, farm type
    💰 Money Starting gold, earnings
    🎒 Inventory Items, tools, weapons
    ⛏️ Skills Farming, Mining, Foraging, Fishing, Combat levels
    💞 Relationships Friendship hearts with NPCs
    🏠 Buildings House upgrades, farm buildings

    How to Use

    1. Find your save file:
    2. Windows: %AppData%/StardewValley/Saves/
    3. macOS: ~/.config/StardewValley/Saves/

    4. Linux: ~/.local/share/StardewValley/Saves/

    5. Back up your save — always make a copy before editing!

    6. Upload to the editor — drag and drop the file

    7. Edit what you want — change money, skills, inventory

    8. Download and replace your save file

    9. Launch the game — your changes will be loaded
