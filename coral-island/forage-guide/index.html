    🌿 Coral Island — Forage & Scavenge Complete Guide

    Foraging and scavenging are your best friends in the early game — free items, easy cash, and essential for museum donations. This database covers every forageable, scavengeable, and underwater collectible in Coral Island.

    🌱 Land Forage

    Items that spawn on the ground across the island. Respawn daily.

    Item Season Location Sell Price Uses
    Wild Horseradish Spring Town, Farm, Forest 30g Cooking, Gift (Jasper 👍)
    Daffodil Spring Town, Forest 20g Gift (Celine ❤️), Crafting
    Leek Spring Forest, Mountain 40g Cooking, Gift (Adeline 👍)
    Morel Spring Forest (rare) 80g Cooking (high-tier dish), Gift (Juniper ❤️)
    Salmonberry Spring Forest bushes (15th–24th) 5g Cooking, Energy source, Gift (Liked by most)
    Spice Berry Summer Forest, Mountain 50g Cooking, Crafting (Spice Berry Wine)
    Summer Spangle Summer Town, Lake area 25g Gift (Celine ❤️), Decoration
    Grape Summer Forest (vine) 40g Cooking, Keg processing
    Sweet Pea Summer Town, Forest 30g Gift (Vera ❤️), Decoration
    Red Mushroom Summer Forest (cave area) 60g Cooking, Gift (Junipter 👍)
    Blackberry Fall Forest bushes 5g Cooking, Energy source
    Hazelnut Fall Forest (under trees) 30g Cooking (roasted), Gift (Balor 👍)
    Chanterelle Fall Forest 80g Cooking, Gift (Juniper ❤️)
    Wild Plum Fall Forest 35g Cooking, Keg processing
    Winter Root Winter Forest, Town 40g Cooking, Gift (March 👍)
    Crystal Fruit Winter Mountain, Forest 80g Cooking, Gift (Eiland ❤️)
    Snow Yam Winter Forest, Town 50g Cooking
    Crocus Winter Town, Forest 25g Gift (Vera ❤️), Dye

    Mussel All Beach (rocks) 15g Cooking, Gift
    Oyster All Beach (pier area) 30g Cooking, Gift (Landen ❤️)
    Sand Dollar All Beach (shoreline) 40g Crafting, Museum
    Sea Urchin Summer, Fall Beach (tide pools) 50g Cooking, Gift
    Starfish All Beach (shallow water) 35g Gift, Museum
    Conch All Beach (pier) 45g Crafting, Museum
    Coral Piece All Beach (reef edge) 25g Crafting, Museum
    Driftwood All Beach (shoreline) 5g Crafting, Building materials
    Seaweed All Beach (water edge) 10g Cooking, Crafting
    Clam All Beach (sand) 25g Cooking, Gift

    💰 Profit Strategy by Season

    🌸 Spring Forage Strategy

    Priority Item Daily Value How to Find
    ⭐ Top Salmonberry (15th–24th) ~200g/day Shake every bush in forest
    ⭐ Top Morel 80g each Check forest daily (rare spawn)
    Good Leek 40g each Forest + Mountain
    Good Wild Horseradish 30g each Town + Farm paths

    ☀️ Summer Forage Strategy

    Priority Item Daily Value How to Find
    ⭐ Top Grape 40g each + Keg value Forest vines — process into wine
    ⭐ Top Red Mushroom 60g each Forest cave area
    Good Spice Berry 50g each + Keg value Forest + Mountain
    Good Summer Spangle 25g each Town paths (gift to Celine)

    🍂 Fall Forage Strategy

    Priority Item Daily Value How to Find
    ⭐ Top Chanterelle 80g each Forest (check daily)
    Good Blackberry 5g each, but massive quantity Bushes — best energy crop
    Good Wild Plum 35g each + Keg value Forest trees
    Good Hazelnut 30g each Under trees in forest

    ❄️ Winter Forage Strategy

    Priority Item Daily Value How to Find
    ⭐ Top Crystal Fruit 80g each Mountain + Forest (rare)
    Good Winter Root 40g each Forest + Town
    Good Snow Yam 50g each Forest
    Good Crocus 25g each Gift to Vera (cheap Loved)

    🎯 Best Forage for Museum Completion

    Item Museum Slot Notes
    Morel Spring Forage Rare spawn — save first one for museum
    Crystal Fruit Winter Forage Only in winter, easy to miss
    Chanterelle Fall Forage Another seasonal must-save
    Sand Dollar Beach Scavenge Common but easy to sell accidentally
    Starfish Beach Scavenge Save first one!
    Conch Beach Scavenge Save for museum bundle
    Brain Coral Underwater Deep zone only
    Purple Coral Underwater Mid zone only
    Abalone Underwater Deep zone only, save for museum

    Data Source: Coral Island v1.0+ official release | Prices may vary slightly with update patches. Forage spawn rates based on community wiki consensus.
