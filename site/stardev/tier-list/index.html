    🏆 全作物 Tier 排名

    数据版本：Stardew Valley 1.6 | 基于每日利润、成长速度、续收能力综合评定

    S 级 — 必种神作

    这些作物是赚钱的绝对核心，无论新手还是老手都推荐优先种植。

    Tier名称说明
    S ⭐ 杨桃 夏季最强作物，基础卖价750金，酿酒后3150金。解锁温室后全年种植，日入26.92金。酿酒流的核心
    S 🍓 草莓 春季之王！蛋节（春13日）买种子，种下后每4天续收。第二年春大量种植，每日利润20.83金
    S 🫐 蓝莓 夏季续收三巨头之一。每4天收获一次，单次收3个，精力效率极高。基础日利润20.80金
    S 🫐 蔓越莓 秋季续收王者。每3天收一次，一次收2个。秋季经济支柱，日利润18.89金

    A 级 — 优秀作物

    虽然不是最强，但配合酿酒/加工后利润可观，或者在特定场景下表现出色。

    Tier名称说明
    A 🌿 啤酒花 酿酒效率之王！11天长成后每天收，酿成淡啤酒卖300金（10倍增值）。温室必备，日利润约16.50金
    A 🎃 南瓜 秋季高价值作物。可巨型作物（收获3-15倍），日利润16.92金。绿洲周四可买种子
    A 🍈 甜瓜 夏季高价值作物。可巨型，酿酒后1050金。日利润14.17金。送礼佳品
    A ☕ 咖啡豆 前期贵但后期神。种一次永久产，每天收。咖啡提供移动速度buff，批量种植后全自动生产线
    A 🥦 花椰菜 春季高价值。可巨型作物，是献祭必要作物。日利润7.92金。巨型时收益爆炸
    A 🍇 葡萄 秋季续收，每3天收一次。酿成葡萄酒后利润可观。日利润16.00金
    A 🌾 小麦 夏季秋季可种。酿成啤酒20倍增值！配合小桶，可用最短时间获得高回报

    B 级 — 中规中矩

    利润尚可，有特定的用途或优势，但在强力作物面前竞争力不足。

    Tier名称说明
    B 🥔 土豆 春季早期赚钱主力。收获时有几率出2-6颗，实际收益波动。日利润约5金，前期推荐
    B 🌶️ 辣椒 夏季续收作物，每3天收一次。是Shane的送礼最爱。日利润10.67金
    B 🍅 番茄 夏季续收，每4天收一次。日利润9.26金。烹饪食谱需求量大
    B 🍆 茄子 秋季续收，每5天收一次。日利润11.20金。适合懒得天天收的玩家
    B 🥬 甘蓝 春季生长快（6天），日利润6.67金。献祭物品，可以快速开场
    B 🥕 萝卜 夏季快速作物（6天），日利润8.33金。酿酒后225金
    B 🫘 山药 秋季作物（10天），日利润10金。献祭必需品，地鼠最爱
    B 🥬 大黄 春季，种子100金（绿洲买），13天成熟。日利润9.23金。可做酒
    B 🌿 苋菜 秋季快速作物（7天），日利润11.43金。能量恢复不错

    C 级 — 不推荐

    利润低、用途少，除非特殊情况（献祭、送礼、装饰）否则不推荐种植。

    Tier名称说明
    C 🌿 防风草 新手入门作物。单价低，日利润仅3.75金。种够献祭所需即可
    C 🌽 玉米 跨夏季和秋季，但日利润仅7.5金。优点是省地，种一次收两季
    C 🌻 向日葵 夏季，种子贵（200金），能产种子但利润低。日利润4金。主要用来产油
    C 🌺 虞美人 夏季花卉，日利润5.71金。主要用于厨师送礼和做幸运午餐
    C 🌹 玫瑰仙子 秋季花卉，种子200金，日利润7.5金。主要用于Jas送礼
    C 🥬 白菜 秋季快速作物（6天），日利润5金。种够献祭即可
    C 🌷 郁金香 春季花卉，日利润1.67金。纯装饰用途，Evelyn最爱
    C 💐 蓝爵士 春季花卉，日利润2.86金。Sandy最爱的礼物，其他用途有限

    Tier 评定标准说明

    等级 日利润范围 说明
    S >18金/天 必种，赚钱核心。优先安排温室+农场主力地块
    A 10-18金/天 优秀，配合加工利润更高。推荐适量种植
    B 5-10金/天 中规中矩，特定场景有用。新手过渡期可以种
    C <5金/天 不推荐纯赚钱，限献祭/送礼/装饰

    💡 关键建议： 解锁小桶（Keg）后优先种杨桃/啤酒花酿酒，赚钱速度翻10倍。温室始终种杨桃+啤酒花。
