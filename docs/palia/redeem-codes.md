## How to Redeem Codes in Palia

1. Launch Palia and enter the game world
2. Press **ESC** to open the main menu
3. Click the **"Redeem Code"** button (or navigate to Settings → Redeem Code)
4. Enter the code exactly as shown (codes are **case-sensitive**)
5. Click **Redeem** — rewards will appear in your inventory

> 💡 **Tip:** Codes usually expire within **7–14 days**, so redeem them as soon as you see them!
