# 🏠 Palia Housing & Decor Complete Guide

房屋系统是 Palia 中最具个性的部分。从帐篷到庄园，你的家影响 Focus 恢复速度和日常生活便利性。

---

## 🏡 住房地块解锁

| 步骤 | 任务/条件 | 说明 |
|:----|:---------|:----|
| 解锁地块 | Welcome to Palia 任务 | Kilima 东北角山上传送门 |
| 建造第一间房 | 100 Sapwood Plank + 35 Stone Brick | 30秒建造 |
| 快速回家 | 地图右上角 Return Home | 30分钟冷却 |

## 🚪 手动添加房间

在 City Hall 购买 **Writs（建筑许可证）** 可解锁更多建筑用地：

| 项目 | 细节 |
|:----|:-----|
| 房间类型 | 标准房、走廊、玄关、阳台 |
| 建筑上限 | 30个建筑件 |
| 布局建议 | 从主体向外扩展成L型或U型 |
| 第二地块 | 建完 Harvest House 后免费获得 |

**额外地块价格**: 第3个 50,000金 → 第4个 75,000金 → 每多一个 +25,000金（最多8个）

## ✨ 装饰加成：Coziness / Harmony

| 属性 | 实际效果 |
|:----|:--------|
| **Coziness（舒适度）** | ✅ **有实际效果**——提升被动 Focus 恢复速度 |
| **Harmony（和谐度）** | ❌ 纯视觉概念——没有游戏内数值加成 |

- 放任何家具都增加 Coziness
- Coziness 上限随房屋等级提升
- 不需要凑套装——前期盲目堆家具即可

> **Tip:** 初期不要纠结套装配色。哪怕放重复的椅子也能堆 Coziness → 加速 Focus 恢复 → 更快练级。

## 📦 存储方案

| 容器 | 槽位 | 获取 |
|:----|:---:|:----|
| 木箱（初始） | 20 | 初始赠送 |
| 铁箱子 | 35 | 铁锭+木料制作 |
| 帕利乌姆箱子 | 50 | 后期制作 |

**整理原则**: 所有箱子内容在烹饪/制作时自动共享。但建议分类：入口（工具）、厨房（食材）、工坊（矿石+木材）、花园（种子）。

> **Tip:** 装饰品（家具/壁纸/地板）**不占存储空间**，可以随便存放。

## 🌿 花园与种植

| 机制 | 说明 |
|:----|:----|
| 种子类型 | 蔬菜、水果、特种作物（详见 [Crops Guide](./crops.md)） |
| 品质 | 普通 / ⭐ Starred（1.5×售价） |
| **交叉授粉** | ❌ 目前 Palia **没有**交叉授粉机制 |
| 肥料 | Quality（-20%生长）/ Rich（+1产量） |
| 洒水器 | 自动浇水，减少每日维护 |
| 多收作物 | 番茄、葡萄——首次收获后再生时间减半 |

**推荐布局**: 4×5 外围种多收作物（番茄/葡萄），中间种高价值单收（姜/蘑菇/卷心菜）。

> **Tip:** 使用 Quality Fertilizer 培育星标食材，烹饪时更容易出星标菜品→售价更高、NPC 更喜欢。
