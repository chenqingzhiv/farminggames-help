# Palia 帕利亚 — 完整攻略

## 游戏概览

- **开发商**: Singularity 6
- **平台**: PC / NS
- **类型**: 种田MMO（免费）
- **特点**: 多人在线种田+冒险

## 新手必做

### 1-10级快速升级

1. **完成新手教程** → 解锁基础工具
2. **种小麦**（3天成熟，烹饪基础）
3. **钓鱼** → 初期最快赚钱方式
4. **采矿** → 升级工具的重要材料

### 赚钱最快方式

| 方式 | 时薪 | 条件 |
|------|------|------|
| 钓鱼（漩涡点） | 3K-5K | 钓鱼技能4+ |
| 稀有虫捕捉 | 2K-4K | 高级捕虫网 |
| 烹饪任务 | 1K-2K | 厨房升级 |
| 种棉花 | 1K-3K | 种子解锁 |
