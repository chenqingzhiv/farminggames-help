# 🏹 Palia Hunting & Bug Catching Complete Guide

Hunting 和 Bug Catching 是 Palia 中获取皮革、肉类、昆虫材料和金币的核心技能。本文涵盖从解锁弓到最佳刷金路线的全部内容。

---

## 🎯 Hunting 解锁

完成主线任务 **Learning the Ropes**（Hassian 发布）后，你将获得第一把 **Makeshift Bow** 和基础箭矢。之后找 Hassian 可以购买高级弓和箭的配方。

| 项目 | 详情 |
|:----|:-----|
| 解锁条件 | 完成 Learning the Ropes 任务 |
| 公会代表 | Hassian |
| 货币 | 狩猎奖章 (Hunting Medal) |
| 技能上限 | Lv 10（继续升级获得成就奖励） |

| 弓 | 等级需求 | 材料 | 特性 |

| Makeshift Bow | Lv 1 | 初始赠送 | 射程短，伤害低 |

| Exquisite Bow | Lv 9 | 5 Flow-Infused Plank + 5 Palium Bar + 3,000金 | 最佳射程，最快拉弓速度 |

### 箭矢类型

| 箭矢 | 效果 | 适用场景 |
|:----|:----|:--------|
| Standard Arrow | 基础伤害 | 普通 Sernuk / Spotted Chapaa |
| Fine Arrow | 高伤害 | 中等以上生物 |
| Slowdown Arrow | 减速目标 | Azure Chapaa / Bluebristle Muujin |
| Dispel Arrow | 破除魔法效果 | Proudhorn / Azure / Bluebristle |

> **Tip:** 背包里同时带 Fine Arrow 和 Dispel Arrow，右键切换。遇到魔法生物先一发 Dispel 再换 Fine 输出。

## 🐞 Bug Catching 完全指南

### 烟雾弹类型

| 类型 | 解锁等级 | 效果 |
|:----|:-------:|:----|
| **Standard Smoke Bomb** | Lv 1 | 基础捕捉，会惊动周围虫子 |
| **Sneaky Smoke Bomb** | Lv 3 | **不会惊动**周围虫子，适合密集区 |
| **Buzzy Jar** | Lv 6 | 显示附近稀有/史诗级虫子的位置（小地图标记） |
| **Honey Lure** | Lv 8 | 在落点生成大量虫子（含稀有） |
| **Smoke Candle** | 消耗品 | 潜行状态下接近虫子不触发逃跑 |

### 最佳捉虫路线

1. **Kilima 夜间路线**: 从喷泉出发 → 图书馆花园 → 农场 → 河边草丛（夜行蛾、萤火虫）
2. **Bahari 白天路线**: Pulsewater Plains → 海边 → 瀑布周边（稀有蝴蝶、宝石甲虫）
3. **蜂蜜诱捕法**: 在 Bahari 选定开阔地 → 使用 Honey Lure → 配合 Sneaky Bomb 连锁捕捉

> **Tip:** 将游戏画质中的 Foliage 调到 Low，地面爬虫更容易发现。配合 Buzzy Jar + Smoke Candle 是捕获史诗虫子的黄金组合。

## 📅 每周 Want 保留清单

每周 NPC 的 Weekly Wants 可能会要求以下狩猎/捉虫产物，**建议保留不要出售**：

- **Sernuk Meat / Chapaa Meat** — 烹饪材料，NPC 常要 cooked meat
- **任何星标虫子** — 多个 NPC 喜爱稀有虫子（Elouisa、Auni）
- **魔法皮革** — 制作高级装备和家具
- **优质皮革 / 厚皮** — 升级装备的必备材料
- **生肉** — 库存保持在 30+ 份

> **Tip:** 周一刷新 Weekly Wants 后，先检查是否有狩猎相关的需求，优先完成 NPC 任务再卖多余材料。
