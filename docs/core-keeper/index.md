# ⛏️ Core Keeper — Complete Guide & Database

**Core Keeper** is a sandbox survival-crafting game by Pugstorm where you explore an underground world, mine resources, farm crops, fight bosses, and build a base. Released in full 1.0 in August 2024, it combines Terraria-style exploration with Stardew Valley-style farming and cooking mechanics.

## 🧭 What You Can Do in Core Keeper

| Activity | Description |
|:---------|:------------|
| ⛏️ **Mining** | Dig through procedurally generated underground biomes for ores, gems, and artifacts |
| 🌱 **Farming** | Plant and water crops, harvest food, cook meals with buffs |
| 🐟 **Fishing** | Catch fish in underground lakes and the Sunken Sea |
| ⚔️ **Combat** | Fight slimes, larva, ghosts, and massive boss Titans |
| 🔧 **Crafting** | Build workbenches, furnaces, anvils, and automate with conveyor belts |
| 👷 **Base Building** | Build walls, floors, furniture, and automation systems |
| 🧪 **Cooking** | Combine ingredients for meals with movement speed, mining, and combat buffs |

## 🗺️ Biome Progression

Core Keeper's underground is divided into biomes in roughly this progression order:

| Biome | Depth | Key Resources |
|:------|:------|:--------------|
| 🟫 **Dirt Biome** | Starting area | Copper, tin, basic slimes |
| 🟥 **Clay Biome** | Surface-ish | Iron, clay, larva |
| 🟩 **Forgotten Ruins** | Dirt+ | Gold, ancient gems |
| 🟦 **Sunken Sea** | Water area | Octarine, pearls, rice, coffee |
| 🟧 **Desert of Beginnings** | Far | Platinum, pineapples, grapes |
| 🟪 **Mold Biome** | Very deep | Scarlett ore, mold, mushrooms |
| 🔥 **Forlorn Metropolis** | Endgame | Galaxite, fire peppers |
| ❄️ **Crystal Biome** | Endgame | Crystallized water |
