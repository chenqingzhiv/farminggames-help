# 🎪 节日活动日历

星露谷物语每年有 **12个主要节日**。每个节日都有独特的活动+奖励。本攻略按季节列出每月重要活动。

---

## 🌸 春季节日

### 春13 — 复活节（蛋蛋节）

| 内容 | 说明 |
|:----|:-----|
| **9AM** 到镇中心广场 | 找刘易斯开始 |
| **活动** | 找彩蛋（9个隐藏彩蛋） |
| **奖励** | 草帽+彩蛋装饰品 |
| **商店** | 草莓种子（100g/包）— **必买！** |

**草莓种子要买多少？**
| 春季第1年 | 买20-30包 | 种完还能收2-3次 |
| 春季第2年+ | 有多少买多少 | 草莓是最赚钱的春季作物 |

### 春24 — 花舞节

| 内容 | 说明 |
|:----|:-----|
| **9AM** 到森林 | 找刘易斯开始 |
| **活动** | 请NPC跳舞 |
| **奖励** | 无直接奖励 |
| **重点** | 如果和NPC好感≥4心，可以一起跳舞 |

**花舞节策略：**
```
1月到3月猛送礼
目标：跟喜欢的NPC好感拉到4心
舞伴：阿比盖尔/海莉/塞巴斯蒂安/艾利欧特最容易
```

|:----|:-----|

| **活动** | 坐潜艇看深海鱼/购物/看美人鱼表演 |


## 🍁 秋季节日

### 秋16 — Stardew Valley博览会

**最重要的节日！**

| 内容 | 说明 |
|:----|:-----|
| **9AM** 到镇中心 | 持续一天 |
| **活动** | 农产品展览+小游戏+购物 |
| **奖励** | 星币（可兑换珍贵物品） |

**如何赢展览（999星币）：**

| 评分项目 | 最佳提交 | 得分 |
|:---------|:---------|:-----|
| 紫色品质最值钱 | 铱星南瓜/铱星果酒 | 极高 |
| 9格全满 | 不同种类物品各放一 | 额外加分 |
| 至少1个动物制品 | 铱星奶酪/蛋黄酱 | 额外加分 |
| 至少1个工匠制品 | 上古果酒/果酱 | 额外加分 |

**最佳展览组合：**
```
1. 铱星南瓜（最高分野菜）
2. 星露谷果酒（酿酒）
3. 钻石（宝石）
4. 五彩碎片（矿物）
5. 铱星奶酪（动物制品）
6. 铱星蛋黄酱（工匠品）
7. 大嘴鲈鱼（鱼）
8. 三叶草/羊肚菌（采集）
9. 任意填满
= 几乎100%拿第1名，得1000星币
```

**星币商店重点购买：**
| 物品 | 价格 | 推荐 |
|:----|:-----|:-----|
| 星果 | 2000星币 | **必买！** 永久+34体力上限 |
| 珍贵茶叶 | 500星币 | 可以种，连续产茶 |
| 农场电脑 | 2000星币 | 看农场数据 |
| 稀有种子 | 100星币 | 种出甜宝石 |

## ❄️ 冬季节日

|:----|:-----|

### 冬15-17 — 夜市（重要！）

| 内容 | 说明 |
|:----|:-----|
| 持续3天 | 海边夜市 |
| 潜艇 | 钓深海鱼（记得每天都去钓！） |
| 商店 | 买稀有种子+咖啡+装饰品 |

**冬天的主要经济来源！** 趁这个机会钓深海鱼卖钱。

| 内容 | 说明 |
|:----|:-----|
| **9AM** 到刘易斯家 | 分享食物+社交 |
| **活动** | 带最贵的料理去比 |
| **奖励** | 无直接奖励，但可以提升全镇好感 |

**带什么去：** 铱星奶酪/铱星果酒/粉红蛋糕（最加分）
