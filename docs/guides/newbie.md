# 种田游戏 — 新手入门指南

## 新手常见问题

### 第一年春天做什么？

1. ✅ 清理农场（树枝/石头/杂草）
2. ✅ 开垦15块地（不要太多，精力不够）
3. ✅ 种防风草（4天成熟，新手期主力）
4. ✅ 钓鱼（初期额外收入）
5. ✅ 和NPC打招呼（为好感铺垫）

### 应该先升级什么？

| 优先级 | 升级 | 原因 |
|--------|------|------|
| 1 | **背包** | 空间不够影响效率 |
| 2 | **浇水壶** | 一次浇更多地 |
| 3 | **锄头** | 开垦更快 |
| 4 | **斧头** | 砍树攒木 |
| 5 | **镐子** | 挖矿采石头 |
