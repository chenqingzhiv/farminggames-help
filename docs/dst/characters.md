#### 🧑‍🔬 Wilson (The Gentleman Scientist)

| Stat | Value |
|:-----|:------|
| **HP** | 150 |
| **Hunger** | 150 |
| **Sanity** | 200 |
| **Unlock** | Starting character (free) |
| **Role** | Utility |

**Special Ability:** Wilson grows a magnificent beard during Winter. The beard provides **insulation** (reducing cold damage) and can be shaved for beard hair, used to craft **Meat Effigy** (revival item). The longer the beard, the better the insulation:

| Beard Stage | Days to Grow | Insulation | Beard Hair |
|:------------|:-------------|:-----------|:-----------|
| Short | 4 | 15 | 1 |
| Medium | 8 | 45 | 3 |
| Long | 16 | 75 | 9 |

**Disadvantage:** None — Wilson is the balanced "vanilla" character, designed as the tutorial survivor.

**Strategy:** Wilson has no stat penalties and the highest starting sanity. Use his beard for winter survival — let it grow fully before Winter begins, then shave it in Spring. The beard hair from a full beard crafts 3 Meat Effigies (with 3 meat + 1 beard hair each). Best choice for learning the game.

**Synergies:** Good with any team. No special team role but no downsides either.

#### 🔥 Willow (The Firestarter)

| Stat | Value |
|:-----|:------|
| **HP** | 150 |
| **Hunger** | 150 |
| **Sanity** | 120 |
| **Unlock** | 640 XP or 1,100 spool |
| **Role** | Utility |

**Special Ability:**
- **Fire Immunity:** Takes no damage from fire (but still from over-heating at high temperatures).
- **Lighter:** Starts with a lighter that never runs out (infinite durability) — provides light and can ignite objects.
- **Sanity from Fire:** Gains +2 sanity per second when near lit fires (campfire, firepit, etc.).

**Disadvantage:** Low sanity (120 base) — more vulnerable to darkness and monster auras. Sanity drops quickly at night without light.

**Strategy:** Use the infinite lighter for early-game light (save resources on torches). Keep a firepit going to passively regain sanity. Craft Bernie early — it's a free tank against shadow creatures and nightmare monsters. Willow excels in the Ruins where sanity management is critical.

**Synergies:** Pairs well with Wormwood (can stand near Wormwood's blooming light). Good for cave exploration teams.

**Synergies:** Excellent with Maxwell (Abigail + shadow puppets = overwhelming force). Good with Wigfrid (Wigfrid tanks, Abigail deals AoE damage).

#### 🪵 Woodie (The Woodchopper)

| Stat | Value |
|:-----|:------|
| **HP** | 150 |
| **Hunger** | 150 |
| **Sanity** | 150 |
| **Unlock** | 640 XP |
| **Role** | Special |

**Special Ability:**
- **Lucy the Axe:** Starts with Lucy, a talking axe that chops trees in 3 hits (vs. 5 for normal axe). Lucy can be recalled to hand (like a boomerang).
- **Werebeaver / Weremoose / Weregoose:** When Woodie's **were-meter** fills (by chopping trees, eating monster meat, or full moon), he transforms. The transformation type depends on the idol carried:
  - **Werebeaver:** Fast chopping/mining, can eat wood for health
  - **Weremoose:** Combat form, higher damage+hp
  - **Weregoose:** Swimming form, fast water travel
- **Idol Crafting:** Can craft Woodie's Idols to force specific transformations.

**Disadvantage:** The were-meter fills at inopportune times — transforming in front of a boss or during a hound wave can be dangerous. In were-form, you lose inventory access and can't use tools.

**Strategy:** Use Lucy for incredibly fast tree farming. Keep an eye on the were-meter bar (visual indicator). Use the transformation offensively — Weremoose is a solid combat form for clearing hounds or taking on medium bosses. The Weregoose form is excellent for ocean exploration in the Return of Them content.

**Synergies:** Great with Winona (catapults cover him while in were-form). Good with Wickerbottom (books + fast wood gathering).

#### ⚔️ Wigfrid (The Performance Artist / Valkyrie)

| Stat | Value |
|:-----|:------|
| **HP** | 200 |
| **Hunger** | 120 |
| **Sanity** | 120 |
| **Unlock** | 960 XP |

**Special Ability:**
- **Battle Spear:** Starts with a spear that deals 42.5 damage (vs. 34 normal).
- **Battle Helm:** Starts with a helmet that absorbs 80% damage (same as Football Helmet, but more durable).
- **Damage Reduction:** Inherently takes 25% less damage from all sources.
- **Battle Born:** Gains sanity and health from killing mobs (sanitary 0.15× damage dealt, health 0.25× damage dealt).

**Disadvantage:** **Strict Carnivore** — only eats meat and meat-based dishes. Cannot eat vegetables, fruits, or honey. This limits food options, especially early game.

**Strategy:** Wigfrid is the best combat character for beginners. The Battle Helm and Battle Spear are massive early-game power spikes. Her damage reduction makes her incredibly tanky. The main challenge is food — you must hunt constantly. Learn to make Meatballs (1 monster meat + 3 fillers) and Pierogi. Kill spiders, frogs, and hounds for food and sanity.

**Synergies:** Best tank for any boss-fighting team. Pairs with Warly (spicy dishes provide extra combat bonuses). Great with Wolfgang (both are frontliners).

| **Hunger** | 175 |

**Synergies:** Amazing with Wendy (Abigail + spider army = massive AoE). Good with Wurt (both are monster-kin but Wurt loses sanity near Webber). Webber + Wortox = infinite soul loops from spiders.

#### 💪 Wolfgang (The Strongman)

| Stat | Value |
|:-----|:------|
| **HP** | 200 |
| **Hunger** | 200 |
| **Sanity** | 200 |
| **Unlock** | 640 XP or 1,100 spool |
| **Role** | Fighter |

**Special Ability:**
- **Mighty Form:** After eating enough (fullness > 150), Wolfgang becomes **Mighty**. In this form:
  - **Damage:** 2.0× damage multiplier (highest in the game)
  - **Carry Capacity:** Can carry heavy objects (statues, boulders) at full speed
  - **Speed:** Slightly faster movement
  - **Mightiness:** Visual indicator — bigger muscles = bigger damage
- **Wimpy Form:** When hungry (< 100 fullness), becomes **Wimpy**:
  - **Damage:** 0.75× damage multiplier
  - **Speed:** Slower
  - **Max HP reduced** (scales with hunger)

**Disadvantage:** **Hunger drain increases with mightiness** — Wolfgang eats 1.5–2× more food than normal characters. Weak form makes early combat dangerous. Managing his form is a constant food pressure.

**Strategy:** Wolfgang is the highest DPS character in the game. Keep hunger above 200 at all times to stay Mighty. Farm large meat sources (koalefants, beefalo herds, hound waves). In Mighty form, you can solo most bosses with kiting. A fully mighty Wolfgang + Dark Sword = 136 damage per hit. Late game: combine with Warly's spicy food for even more damage.

**Synergies:** Best with Warly (spicy Volt Goat Chaud-Froid = even more damage). Good with Wigfrid (dual frontliners). Needs a farmer/hunter to supply food (Wickerbottom, Wormwood).

#### 🤖 WX-78 (The Soul-less Automaton)

| Stat | Value |
|:-----|:------|
| **Base HP** | 100 |
| **Base Hunger** | 150 |
| **Base Sanity** | 150 |
| **Max Upgraded** | 400 HP / 300 Hunger / 300 Sanity |

**Special Ability:**
- **Gear Upgrades:** Eats gears to permanently increase stats. Each gear gives +20 max HP, +6.25 max Hunger, +6.25 max Sanity (caps at 15 gears).
- **Lightning Charge:** Getting struck by lightning (or using a Lightning Rod) provides:
  - Speed boost
  - Night vision (glowing eyes)
  - Light emission
  - Sanity boost
- **Circuit System (DST Rework):** Can install circuits with charged circuits for various bonuses (speed, regen, processing).

- **No Sanity from Clothing:** Does not gain sanity from wearing fashionable clothing.

**Disadvantage:** **Slow in rain** — movement speed reduced by 50% when wet. Takes damage from standing in water (puddles, ocean). Low base stats require gear investment. Cannot gain sanity from clothing.

**Strategy:** Rush to find clockwork mobs (chess pieces) or tumbleweed gears. Prioritize getting 6–8 gears for a solid stat boost. Use a Rain Hat or Eyebrella to counter rain penalties. Lightning rods near base = free speed boost during storms. The circuit system adds significant versatility.

**Synergies:** Good with Wickerbottom (gears from clockwork knights). Pairs with Winona's generators for circuit charging.

#### 🎩 Maxwell (The Gentleman Shadow Mage)

| Stat | Value |
|:-----|:------|
| **HP** | 75 |
| **Hunger** | 75 |
| **Sanity** | 200 |
| **Unlock** | 1,024 XP or 1,500 spool (adventure mode completion) |
| **Role** | Summoner |

**Special Ability:**
- **Shadow Codex:** Starts with the Codex Umbra, which allows him to summon **Shadow Puppets**:
  - **Duelist:** Chopping/mining puppet (costs 20 sanity, drains 1 sanity/20s)
  - **Miner:** Mining puppet (same cost)
  - **Warrior:** Combat puppet (costs 40 sanity, drains 1 sanity/8s)
- **Natural Sanity Regeneration:** Recovers 6.6 sanity/min naturally (3× normal rate). Sanity regenerates even in darkness.
- **Shadow Insight:** Can see in the dark (minimal light radius always active).

**Disadvantage:** **Very low HP (75) and Hunger (75).** One-two hits from most enemies can kill him. His puppets drain sanity quickly if over-summoned, leading to nightmare creatures.

**Strategy:** Maxwell is a glass cannon commander. Let your shadow puppets do the work while you stay at range. Keep 3–4 duelists for wood/farming and 1–2 warriors for combat. Puppets don't follow through wormholes. Important: Maxwell's sanity regen stops when Codex is closed. Use the Codex strategically — open to generate puppets, close to save sanity.

**Synergies:** Amazing with Wendy (both summoner types). Pairs well with Wickerbottom (books + puppets). Needs a meat shield — Wigfrid or Wolfgang to draw aggro.

| Stat | Value |
|:-----|:------|
| **HP** | 150 |
| **Hunger** | 150 |
| **Sanity** | 250 |
| **Unlock** | 640 XP |
| **Role** | Support |

The no-sleep restriction means you must always have sanity recovery options (cooked green caps, taffy, cooked cactus flesh).

**Synergies:** Perfect with Wormwood (books grow Wormwood's plants). Pairs with Wolfgang (fast tech, Wolfgang protects). Incredible with Winona (catapult + tentacle field = death zone).

## 🏆 Character Recommendations

### 🟢 Best Characters for Beginners

| Character | Why? |
|:----------|:-----|
| **Wilson** | No downsides, balanced stats, beard teaches seasonal preparation. The definitive starter character. |
| **Wendy** | Abigail does the fighting for you. Reduced sanity loss means fewer shadow monster problems. Learn the game safely. |

**Best Starter Pick:** **Wilson** — the tutorial character for a reason. Simple, balanced, teaches all core mechanics.

| Character | Why? |
|:----------|:-----|
| **Wendy** | Abigail provides DPS and distraction. Wendy's lower damage matters less when you control the minion. Great for solo bossing. |
| **Wolfgang** | Highest DPS in the game. Good kiting and you can solo even raid bosses with practice. Food pressure is manageable alone. |
| **WX-78** | Gear upgrades make you a stat monster. Lightning charge for speed+vision. Circuit system for versatility. Self-sufficient. |

**Best Solo Pick:** **Wendy** — Abigail enables safe exploration and combat. The AoE damage clears hound waves efficiently. Less reliance on base infrastructure.

**Best Team Composition (4-player):** **Warly** (buffer) + **Wolfgang** (DPS) + **Wigfrid** (tank) + **Wortox** (healer). This team can handle any boss in the game with proper coordination.

function getDifficultyClass(d) {
  if (d === "Easy") return "diff-easy";
  if (d === "Medium") return "diff-medium";
  return "diff-hard";
}

function getStatClass(val) {
  if (val >= 200) return "stat-high";
  if (val >= 130) return "stat-mid";
  return "stat-low";

function renderTable() {
  const search = document.getElementById("char-search").value.toLowerCase();

  const filtered = characters.filter(c => {

    return true;
  });
  
  tbody.innerHTML = filtered.map(c => {

    const sanityClass = getStatClass(c.sanity);

let sortDir = {};
function sortTable(col) {
  const tbody = document.querySelector("#char-table tbody");
  const rows = Array.from(tbody.querySelectorAll("tr"));
  sortDir[col] = !sortDir[col];
  const dir = sortDir[col] ? 1 : -1;
  
  rows.sort((a, b) => {
    const va = a.cells[col].textContent.trim().toLowerCase();
    const vb = b.cells[col].textContent.trim().toLowerCase();
    const na = parseFloat(va.replace(/,/g,''));
    const nb = parseFloat(vb.replace(/,/g,''));
    if (!isNaN(na) && !isNaN(nb)) return (na - nb) * dir;
    return va.localeCompare(vb) * dir;
  });
  
  rows.forEach(r => tbody.appendChild(r));
  
  // Update sort indicator
  const indicators = ["Character", "HP", "Hunger", "Sanity", "Difficulty", "Role", "Special Ability", "Disadvantage"];
  document.getElementById("sort-indicator").textContent = indicators[col] + " " + (sortDir[col] ? "↓" : "↑");
}
