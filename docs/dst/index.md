# 🌲 Don't Starve Together — Farming & Survival Guide

**Don't Starve Together** (DST) is a multiplayer survival game by Klei Entertainment. Set in a dark, whimsical wilderness, it challenges you to manage hunger, sanity, and health while exploring, crafting, and building a base. The 2022 gardening overhaul added one of the deepest farming systems in any survival game.
