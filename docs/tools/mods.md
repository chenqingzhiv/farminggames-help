# Stardew Valley Mods Guide

Mods can completely transform your Stardew Valley experience — from visual upgrades to massive content expansions. Here's our curated collection of the best mods, organized by category.

---

## ⚙️ How to Install Mods

### Prerequisites

1. **Install SMAPI** (Stardew Modding API) — [Download here](https://smapi.io/)
2. Extract SMAPI to your Stardew Valley folder
3. Run `install.exe` on Windows (or terminal on Mac/Linux)
4. Launch the game through SMAPI (a new shortcut will be created)

### Installing a Mod

1. Download the mod from Nexus Mods
2. Extract the ZIP file
3. Copy the mod folder into `Stardew Valley/Mods/`
4. Launch the game via SMAPI

### Mod Manager (Recommended)

[**Stardrop**](https://github.com/Floogen/Stardrop) — an open-source, cross-platform mod manager:
- Auto-install and update mods
- Enable/disable mods with one click
- Manage mod profiles (different sets for different saves)

## 📦 Starter Mod Pack

**For new players, install these 5 mods first:**

1. **SMAPI** — Required for all mods
2. **UI Info Suite 2** — See everything you need
3. **Lookup Anything** — Never search the wiki again
4. **Chests Anywhere** — Organize your storage
5. **Automate** — Let machines work while you explore

This pack gives you quality-of-life improvements without changing the core game experience.

---

## ⚠️ Important Notes

- **Back up your save files** before installing mods
- **Check mod compatibility** — some mods don't work together
- **Update regularly** — game updates can break mods
- **SMAPI tells you** which mods need updates when you launch
- **Multiplayer**: All players need the same mods installed
