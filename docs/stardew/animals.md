## 🏗️ Ranching Strategy by Game Phase

### Early Game (Year 1, Spring-Summer)

```
Priority: Chickens → Mayonnaise Machine
Investment: 800g chicken + 4,000g coop + 4,000g machine
Daily Income: 304g/day per chicken
Strategy: Start with 4 chickens, max out mayonnaise production
```

### Mid Game (Year 1, Fall-Winter)

```
Priority: Add Pigs (Fall only) + Goats
Investment: 1,600g pig + 2,500g goat
Daily Income: 725g/day (pig) + 459g/day (goat cheese)
Strategy: Maximize Fall pig profits, transition to goats in Winter
```

### Late Game (Year 2+)

```
Priority: Dinosaurs + Bee Houses + Truffle Oil
Investment: 500g dinosaur egg + 2,500g bee hives
Daily Income: 800g/day (dino) + 156g/day (bee) + 725g/day (pig Fall)
Strategy: Diversify. Dinosaurs for steady income, pigs for Fall spikes
```

|| Happiness | Indicator | Effect |
|:---|:---|:---|:---|

## 💡 Ranching Tips

1. **Pigs in Fall are unbeatable** — Place them near trees, let them roam, collect truffles daily. One pig can earn 725-1,250g/day.
2. **Goat cheese > Cow cheese** — The 1.7x markup on goat milk beats the 1.2x on cow milk. Prioritize goats once you can afford them.
3. **Dinosaur mayonnaise is the best daily artisan product** — 800g/day with no inputs after purchase. ROI in less than 1 day.
4. **Auto-petter is worth it for large ranches** — At 8+ animals, the time saved petting adds up to hours per week.
5. **Bee hives near flowers** — Place hives next to fairy roses, poppies, or spring flowers for maximum honey output.
6. **Rabbits are niche** — Only keep for Lucky Rabbit's Foot quests. Poor ROI compared to chickens.
7. **Sheep are the worst animal** — 4-day shearing cycle + low cloth markup. Skip unless you need wool.
8. **Gold Watering Can for animals** — Increases chance of gold-quality products. Essential for max profits.
