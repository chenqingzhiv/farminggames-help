# 🌟 Stardew Valley Skills Guide — Best Skill Tree Choices

> Stardew Valley has **5 core skills**, each granting a specialization choice at level 5 and a master specialization at level 10. This guide breaks down every option so you can make the best pick.

*Data sourced from Stardew Valley Wiki v1.6 | 🏆 marks recommended builds*

---

## 🧑‍🌾 Farming

Farming is the heart of Stardew Valley. It affects crop efficiency and artisan good value.

### Level-Up Unlocks

| Level | Unlock | Details |
|:------|:-------|:--------|
| 1 | 🧹 **Scarecrow** | Scares crows away from your crops |
| 2 | 🪨 **Stone Fence**、🌱 **Basic Fertilizer** | Basic Fertilizer boosts crop quality |
| 3 | 🐝 **Bee House**、🧪 **Speed-Gro**、🥪 Farmer's Lunch recipe | Bee House for honey, Speed-Gro for faster growth |
| 4 | 💧 **Basic Sprinkler**、🫙 **Preserves Jar** | Basic Sprinkler (4 tiles), Preserves Jar for jams/jellies |
| 5 | **Choose specialization: Rancher 🆚 Tiller** | See below |
| 6 | 💧 **Quality Sprinkler**、🛢️ **Keg**、🧀 **Cheese Press**、🪵 **Hardwood Fence** | Quality Sprinkler (8 tiles), Keg for wine — core money maker |
| 7 | 🧶 **Loom** | Rabbit's wool / Sheep wool → Cloth |
| 8 | 🛢️ **Oil Maker**、🌱 **Seed Maker**、🔩 **Iron Fence** | Oil Maker, Seed Maker for crop propagation |
| 9 | 🌿 **Quality Fertilizer**、🔥 **Cookout Kit**、⚡ **Deluxe Speed-Gro** | High-tier fertilizer, outdoor cooking, fastest growth |
| 10 | **Choose master specialization** | See below |

### Specialization Choices

| Level | Specialization | Effect | 🏆 Recommendation |
|:------|:---------------|:-------|:-----------------|
| **⭐ Lv5** | **Rancher** 🐄 | Animal products value +20% | Ranch-focused |
| | **Tiller** 🌾 | Crop value +10% | **🏆 Recommended (Artisan path)** |
| **⭐ Lv10 (Rancher →)** | **Coopmaster** 🐔 | Coop animal products +20%, incubation time halved | Coop-focused |
| | **Shepherd** 🐑 | Barn animal products +20%, shearing faster | Barn-focused |
| **⭐ Lv10 (Tiller →)** | **Artisan** 🍷 | Artisan goods (wine/cheese/cloth, etc.) value +40% | **🏆 Best pick (highest profit)** |
| | **Agriculturist** 🌱 | Crops grow 10% faster | Speed farming |

### 💡 Recommended Builds

> **🏆 Standard Route: Tiller → Artisan**
> The Artisan +40% bonus is the strongest profit multiplier in the game. Starfruit wine, truffle oil, and aged cheeses all get massive value boosts.
>
> **Ranch Route: Rancher → Shepherd**
> If you're focusing on pigs for truffles, this route rivals Artisan in profitability.

## 🎣 Fishing

Fishing is the most profitable early-game skill and one of the fastest to level.

### Level-Up Unlocks

| Level | Unlock | Details |
|:------|:-------|:--------|
| 1 | 🎣 **Training Rod** (25g)、🪱 **Bait** recipe | Training Rod for easier fishing, Bait crafting |
| 2 | 🎣 **Fiberglass Rod** (1,800g) | Rod that supports bait attachments |
| 3 | 🪤 **Crab Pot**、🥘 Dishes of the Sea recipe | Crab Pot fishing, Dishes of the Sea meal |
| 4 | ♻️ **Recycling Machine** | Recycles trash into Refined Quartz / Cloth / Coal |
| 5 | **Choose specialization: Fisher 🆚 Gatherer** | See below |
| 6 | 🌀 **Spinner**、🪤 **Trap Bobber**、🪱 **Bait Maker** | Spinner reduces bite time, Trap Bobber slows fish escape |
| 7 | 🪤 **Cork Bobber** | Increases fishing bar size |
| 8 | 💎 **Treasure Hunter**、🪱 **Worm Bin** | Increases treasure chest chance, auto-produces bait |
| 9 | 🧲 **Magnet**、🌀 **Dressed Spinner** | Magnet boosts treasure chests, Dressed Spinner for faster bites |
| 10 | **Choose master specialization** | See below |

### Specialization Choices

| Level | Specialization | Effect | 🏆 Recommendation |
|:------|:---------------|:-------|:-----------------|
| **⭐ Lv5** | **Fisher** 🐟 | Fish sell price +25% | **🏆 Recommended (sell fish)** |
| | **Gatherer** 🪤 | Crab Pot yield doubles | Crab Pot automation |
| **⭐ Lv10 (Fisher →)** | **Angler** 🎣 | Fish sell price +50% (total +75%) | **🏆 Best pick** |
| | **Pirate** 💰 | Treasure chest chance doubles | Treasure hunter |
| **⭐ Lv10 (Gatherer →)** | **Mariner** 🚢 | Crab Pots no longer catch trash | Clean Crab Pots |
| | **Luremaster** 🪤 | Crab Pots no longer need bait | **🏆 Best for Crab Pot automation** |

### 💡 Recommended Builds

> **🏆 Standard Route: Fisher → Angler**
> +75% fish sell price. Legend fish can sell for 15,000g+.
>
> **🏆 Crab Pot Automation (1.6 recommended): Gatherer → Luremaster**
> No bait needed + doubled output. Pairs perfectly with automation mods for passive income.

## 🌿 Foraging

Foraging affects wild pickup quality, wood yield, and tree products.

### Level-Up Unlocks

| Level | Unlock | Details |
|:------|:-------|:--------|
| 1 | 🥗 **Field Snack**、🌱 **Wild Seeds (Spring)** | Field Snack for energy, Spring Wild Seeds |
| 2 | 🥪 **Survival Burger** recipe | Survival Burger (energy restore) |
| 3 | 🪚 **Tapper**、🌱 **Wild Seeds (Summer)** | Tapper for maple syrup / oak resin / pine tar |
| 4 | 🔥 **Charcoal Kiln**、🌱 **Wild Seeds (Fall)** | Charcoal Kiln (10 Wood → 1 Coal) |
| 5 | **Choose specialization: Forester 🆚 Gatherer** | See below |
| 6 | ⚡ **Lightning Rod**、🌱 **Wild Seeds (Winter)**、🍄 **Mushroom Log** 🆕1.6 | Lightning Rod, Winter Wild Seeds, Mushroom Log |
| 7 | 🌲 **Tree Fertilizer** | Speeds up tree growth |
| 8 | 🎋 **Wood Chipper** 🆕1.6 | Hardwood → Wood conversion |
| 9 | 🥣 **Cookout Kit**、🍳 **Deluxe Speed-Gro (for Wild Seeds)** | Outdoor cooking kit |
| 10 | **Choose master specialization** | See below |

### Specialization Choices

| Level | Specialization | Effect | 🏆 Recommendation |
|:------|:---------------|:-------|:-----------------|
| **⭐ Lv5** | **Forester** 🪓 | Wood from trees +25% | Heavy lumberjack |
| | **Gatherer** 🍄 | Foraged items doubled | **🏆 Recommended (highest value)** |
| **⭐ Lv10 (Forester →)** | **Lumberjack** 🪵 | Trees drop Hardwood occasionally | Good when hardwood is scarce |
| | **Tapper** 🍁 | Tapper products value +25% | Oak resin / syrup farming |
| **⭐ Lv10 (Gatherer →)** | **Botanist** 🌸 | Foraged items always Iridium quality | **🏆 Best pick (doubled + Iridium quality)** |
| | **Tracker** 🗺️ | Shows foraged item locations on minimap | Helpful for beginners |

### 💡 Recommended Builds

> **🏆 Standard Route: Gatherer → Botanist**
> Double foraged items + automatic Iridium quality is one of the strongest passive skills in the game. A single blackberry becomes Iridium quality AND doubled for serious profit.
>
> **Lumberjack Route: Forester → Lumberjack**
> If you're expanding rapidly in year one and need massive amounts of wood and hardwood.

## ⛏️ Mining

Mining affects ore yield and gem value.

### Level-Up Unlocks

| Level | Unlock | Details |
|:------|:-------|:--------|
| 1 | 🗡️ **Miner's Treat** recipe | Miner's Treat (energy restore) |
| 2 | 💣 **Cherry Bomb** | Cherry Bomb |
| 3 | ⛏️ **Pickaxe Upgrade Needed**、📖 **Mining Journal** | Copper pickaxe upgrade unlocks more content |
| 4 | 🔥 **Furnace**、🪨 **Sturdy Ring** | Furnace, Sturdy Ring (knockback immunity) |
| 5 | **Choose specialization: Miner 🆚 Geologist** | See below |
| 6 | 💣 **Bomb**、🛡️ **Slime Charmer Ring** 🆕 | Bomb, Slime Charmer Ring (slime damage immunity) |
| 7 | 🧱 **Staircase**、🔬 **Geode Crusher** 🆕1.6 | Staircase, Geode Crusher |
| 8 | ⚡ **Explosive Ammo**、🔩 **Mega Bomb** | Explosive Ammo, Mega Bomb |
| 9 | 💎 **Diamond**、🛡️ **Protective Ring** 🆕 | Diamond (Crystalarium), Protective Ring |
| 10 | **Choose master specialization** | See below |

### Specialization Choices

| Level | Specialization | Effect | 🏆 Recommendation |
|:------|:---------------|:-------|:-----------------|
| **⭐ Lv5** | **Miner** ⛏️ | +1 ore per node (Copper/Iron/Gold) | **🏆 Recommended (ore farming)** |
| | **Geologist** 💎 | Gems found in pairs | Gem-focused |
| **⭐ Lv10 (Miner →)** | **Blacksmith** 🔨 | Metal bars sell for +50% | **🏆 Recommended (early money)** |
| | **Prospector** 💎 | Coal found twice as often | Coal shortage |
| **⭐ Lv10 (Geologist →)** | **Gemologist** 💠 | Gems sell for +30% | Crystalarium gem farming |
| | **Excavator** 🧱 | Artifacts found in pairs | Museum collection |

### 💡 Recommended Builds

> **🏆 Standard Route: Miner → Blacksmith**
> Sell metal bars for +50% profit early game. Can earn 50,000g+ in the first year.
>
> **🏆 Late-Game Route: Miner → Prospector**
> Coal demand skyrockets in 1.6 (smelting / crafting). Prospector massively relieves coal shortages.
>
> **Gem Route: Geologist → Gemologist**
> Pair with Crystalarium for gem replication. Diamonds sell for 900g base (+30% = 1,170g). Great passive income.

## ⚔️ Combat

Combat affects survivability and monster drops. Essential for mining deeper levels.

### Level-Up Unlocks

| Level | Unlock | Details |
|:------|:-------|:--------|
| 1 | 🥗 **Roots Platter** recipe | Roots Platter (+3 Attack) |
| 2 | 🗡️ **Life Elixir** recipe | Life Elixir (full HP restore) |
| 3 | 🥾 **Various Boots available in shops** | More boots available for purchase |
| 4 | 🗡️ **Weapon Upgrades Available**、🍳 **Stir Fry** recipe | More weapon options unlocked |
| 5 | **Choose specialization: Fighter 🆚 Scout** | See below |
| 6 | 🗡️ **More Weapons**、🛡️ **Slime Charmer Ring** | More weapons available |
| 7 | 🧪 **Squid Ink Ravioli** 🆕1.6、🍝 **Mushroom Soup** | Squid Ink Ravioli (+1 Defense), Mushroom Soup |
| 8 | 🛡️ **More Rings**、🍳 **Algae Soup** | More ring unlocks |
| 9 | 🍲 **Sashimi**、🥣 **Maki Roll** (if not done at Qi) | Sashimi recipe |
| 10 | **Choose master specialization** | See below |

### Specialization Choices

| Level | Specialization | Effect | 🏆 Recommendation |
|:------|:---------------|:-------|:-----------------|
| **⭐ Lv5** | **Fighter** ⚔️ | Attack +15%, HP +15 | **🏆 Recommended (general use)** |
| | **Scout** 🎯 | Critical hit chance +50% | Crit build |
| **⭐ Lv10 (Fighter →)** | **Brute** 💪 | Attack +15% more (total +30%) | **🏆 Best pick (raw damage)** |
| | **Defender** 🛡️ | HP +25 more (total +40 HP) | Survival build |
| **⭐ Lv10 (Scout →)** | **Acrobat** 🤸 | Special move cooldown -50% | Skill spam build |
| | **Desperado** 🔫 | Critical hits are instant kills | **🏆 Recommended for crit builds** |

### 💡 Recommended Builds

> **🏆 Standard Route: Fighter → Brute**
> Simple +30% attack damage boost that works in every situation. Pairs with the Galaxy Sword for 100+ per hit.
>
> **🏆 Crit Build: Scout → Desperado**
> +50% crit chance + instant kill on crits. Devastating against high-HP enemies in the Skull Cavern. Best paired with the 🐉 Dragontooth Cutlass or Yeti Tooth.
>
> **Defense Build: Fighter → Defender**
> Great for beginners or Hard Mode mines. Combine with an Iridium Band + Crab Shell Ring for maximum survivability.
