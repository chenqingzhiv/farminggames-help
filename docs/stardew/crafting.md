## 🏭 Production Chains

### Wine Production (Keg)

```
Fruit → Keg (7 days) → Wine (3x base price)

Starfruit → Starfruit Wine: 750g → 2,250g (4.2x with Artisan = 3,150g)
Ancient Fruit → Ancient Fruit Wine: 550g → 1,650g (4.2x = 2,310g)
Melon → Melon Wine: 250g → 750g (4.2x = 1,050g)
Pumpkin → Pumpkin Juice: 320g → 720g (2.25x = 1,620g)
Blueberry → Blueberry Wine: 50g → 150g (4.2x = 210g)
Cranberry → Cranberry Wine: 75g → 225g (4.2x = 315g)
Strawberry → Strawberry Wine: 120g → 360g (4.2x = 540g)
```

### Beer Production (Keg)

```
Wheat → Keg (1 day) → Beer: 10g → 200g (20x!)

Hops → Keg (1.5 days) → Pale Ale: 25g → 300g (12x!)
```

### Animal Product Chains

```
Cow Milk → Cheese Press (4hrs) → Cheese: 190g → 230g (1.2x)
Cow Milk → Mayonnaise Machine (1 day) → Mayonnaise: 190g → 190g (1x)
Chicken Egg → Mayonnaise Machine (1 day) → Mayonnaise: 50g → 190g (3.8x!)
Duck Egg → Mayonnaise Machine (1 day) → Duck Mayo: 95g → 275g (2.9x)
Goat Milk → Cheese Press → Goat Cheese: 225g → 375g (1.7x)
Sheep Wool → Cloth Press → Cloth: 340g → 470g (1.4x)
Dinosaur Egg → Mayonnaise Machine → Dinosaur Mayo: 350g → 800g (2.3x)
```

### Seed Making (Seed Maker)

```
Crop → Seed Maker → Seeds

Ancient Fruit → 1.34 seeds avg (repeat forever)
Starfruit → 0.83 seeds avg
Blueberry → 0.83 seeds avg (3 berries per harvest)
Pumpkin → 0.83 seeds avg
Cranberry → 0.83 seeds avg (2 berries per harvest)
Hop → 0.83 seeds avg (daily regrow)
```

## 💡 Crafting Tips

1. **Keg is king** — Always prioritize kegs over preserves jars. Wine profit is 2-3x jelly.
2. **Seed Maker for Ancient Fruit** — The single best investment after your first ancient seed. Turns one seed into an infinite income stream.
3. **Beer has the highest multiplier** — Wheat grows in 4 days, costs 10g, and beer sells for 200g. That's a 20x return.
4. **Mayonnaise is the best early-game income** — Chickens are cheap, eggs are free, and mayo sells for 190g.
5. **Crystalarium for gems** — Dump duplicate diamonds/emeralds into a crystalarium for passive gem income.
6. **Super Bomb for mining** — 5x faster than a pickaxe for clearing floors. Essential for Skull Cavern.
7. **Gold Watering Can** — Buy from the Wizard after donating the first prismatic shard. Guarantees iridium-quality crops.
8. **Repair Kits** — Always carry basic repair kits when going deep into mines. A broken pickaxe = game over.
