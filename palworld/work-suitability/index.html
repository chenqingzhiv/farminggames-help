    ⚡ Palworld — Pal Work Suitability Database

    This database lists every Pal's work suitability skills — what jobs they can do at your base and at what level. Use this to optimise your base automation.

    Quick Navigation: - Understanding Work Suitability - 🔥 Kindling - 💧 Watering - 🌱 Planting - ⚡ Generating Electricity - 🔧 Handiwork - 📦 Transporting - 🪓 Lumbering - ⛏️ Mining - ❄️ Cooling - 🌾 Farming - 🧹 Gathering - Medicine Production

    📐 Understanding Work Suitability

    Each Pal has a Work Suitability rating (1–5) for specific job types. Higher numbers mean: - Faster work speed - Access to higher-tier workstations - Better quality output

    Formula: Work speed (ticks) = Base Speed × Work Suitability Level × Pal Level Bonus

    When assigning Pals to base tasks, a single high-level specialist often outperforms several low-level generalists. But for large bases, multiple Pals can work in parallel.
